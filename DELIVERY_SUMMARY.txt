COMPILATION COMMANDS PROVIDED
================================================================================

Option 1 - System CLIPS Library:
  g++ -std=c++17 -Wall -Wextra -O2 \
      src/main.cpp src/InferenceEngine.cpp \
      -lclips -lm -o diabetes_expert_system

Option 2 - CLIPS Source Files:
  g++ -std=c++17 -Wall -Wextra -O2 \
      src/main.cpp src/InferenceEngine.cpp \
      external/clips/core/*.c \
      -Iexternal/clips/core -lm \
      -o diabetes_expert_system

Option 3 - Using Makefile:
  make              # Auto-detect
  make with-source  # Use source files
  make test         # Build and test
